TypeScript基本使用

基础类型

  • 布尔值，如isDone: boolean
  • 所有数值，如total: number
  • 字符串，如name: string; 模板字符串的使用，${变量名}，效果等于 + 变量名… +
  • 数组，两种方式：
    1.list: number[], arr: string[]
    2.list: Array, arr:Array
  • 元组，类型不必相同的元素数组，如let x: [number,string,string] ; x=[1,2,3]//Error
  • 枚举，如enum Color {Red, Green, Blue}, let c: Color = Color.green;let name:string=Color[1];//name 为’Green’
  • any, 如aaa:any = ‘bbb’
  • void, 表示函数无返回值，如aaa():void{}
  • null, 如n:null
  • undefined, 如u:undefined
    默认情况下null和undefined是所有类型的子类型。 就是说你可以把 null和undefined赋值给number类型的变量。然而，当你指定了–strictNullChecks标记，null和undefined只能赋值给void和它们各自

  • never, 表示函数返回永不存在的值，如function aaa(): never{}

    类型断言

    知道某个变量是某种类型，用<>或as表示
    如let num:number = (aaa).length 或 let num:number = (aaa as string).length
